# Advanced Examples

This folder contains more complex agent patterns grouped by theme.

## Folder Guide

- `agentic_patterns/`: agentic RAG, supervisor teams, dynamic tool creation, and Streamlit dashboards.
- `self_correcting_rag_workflows/`: self-grading and hybrid local-plus-web RAG workflows.
- `planning_workflows/`: planners, replanners, and budget-guard patterns.
- `mcp_basics/`: a minimal MCP client/server example.
- `multi_server_mcp/`: orchestration across multiple MCP servers.
- `prompt_experiments/`: prompt routing and self-improving agent patterns.
- `evaluation/`: evaluation and scoring workflows.
